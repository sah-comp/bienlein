Converter.

Converter. To add your own converter simply add a php file to the converter directory of your Cinnebar installation. Name the converter after the scheme Converter_* extends Cnoverter and implement a convert() method. As an example see Converter_MySqlDate. Example usage of the MySqlDate converter:

  1.  <?php
  2.  $mysqldateconverter = new Converter_MySqlDate();
  3.  $mysqldate = $mysqldateconverter('23.03.1984');
  4.  ?>
