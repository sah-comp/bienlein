getQueriesFromSQLFile parses a sql file and extracts all queries for further processing with pdo execute. Taken from http://jakoch.de/2012/02/27/php-importsql-und-getqueriesfromsqlfile/.

  • strips off all comments, sql notes, empty lines from an sql file
  • trims white-spaces
  • filters the sql-string for sql-keywords
  • replaces the db_prefix is not used, but can be uncommented

  • return: Trimmed array of sql queries, ready for insertion into db.
