Class Sessionhandler

Description
Description | Descendents | Methods (details)

Manages a user session.

Manages a user session. To add your own sessionhandler simply add a php file to the sessionhandler directory of your Cinnebar installation. Name the sessionhandler after the scheme Sessionhandler_* extends Cinnebar_Sessionhandler and implement all methods seen here. You can than use it by defining it as your default sessionhandler in your configuration file config.example.php. As an example see Sessionhandler_Apc.
