Validates a value against a validation rule.

Validates a value against a validation rule. To add your own validator simply add a php file to the validator directory of your Cinnebar installation. Name the validator after the scheme Validator_* extends Validator and implement a validate() method. As an example see Validator_Range. Example usage of the range validator:

  1.  <?php
  2.  $range = new Validator_Range(array('min' => 1, 'max' => 100));
  3.  if ($range->validate(77)) echo '1 >= 77 <= 100 is true';
  4.  if ($range->validate(177)) echo '1 >= 177 <= 100 is false';
  5.  ?>
