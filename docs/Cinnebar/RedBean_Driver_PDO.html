		runQuery (line 242)
	
 
	
	

This method runs the actual SQL query and binds a list of parameters to the query.


This method runs the actual SQL query and binds a list of parameters to the query. slots. The result of the query will be stored in the protected property $rs (always array). The number of rows affected (result of rowcount, if supported by database) is stored in protected property $affected_rows. If the debug flag is set this function will send debugging output to screen buffer.
